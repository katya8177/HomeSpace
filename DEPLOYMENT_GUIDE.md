## Шаг 6: Протестировать функционал

### Тест редактирования:
1. Откройте семейный чат
2. Напишите сообщение
3. Наведите на сообщение → появится кнопка "⋮"
4. Нажмите → выберите "✎ Редактировать"
5. Измените текст
6. Проверьте что добавилась отметка "(ред.)"

### Тест удаления (для себя):
1. Напишите новое сообщение
2. Нажмите на кнопку "⋮"
3. Выберите "🗑 Удалить у себя"
4. Подтвердите
5. Сообщение должно исчезнуть из вашего чата
6. В других браузерах/устройствах сообщение должно остаться

### Тест удаления (для всех):
1. Напишите новое сообщение
2. Нажмите на кнопку "⋮"
3. Выберите "🗑 Удалить для всех"
4. Подтвердите
5. Сообщение должно исчезнуть у всех членов семьи

### Тест прав доступа:
1. Залогиньтесь как обычный пользователь
2. Напишите сообщение другого пользователя
3. Нажмите на кнопку "⋮"
4. Кнопка "🗑 Удалить для всех" должна быть СЕРОЙ (неактивной)
5. Можно только "🗑 Удалить у себя"

6. Залогиньтесь как родитель
7. Нажмите на кнопку "⋮" у сообщения другого пользователя
8. Кнопка "🗑 Удалить для всех" должна быть БЕЛОЙ (активной)

## Проблемы и решения

### Проблема: Кнопка "⋮" не появляется
**Решение:**
1. Проверьте F12 → Console на ошибки
2. Перезагрузите страницу (Ctrl+F5)
3. Убедитесь что обновили файл ChatScene.js
4. Перезагрузите сервер Node.js

### Проблема: Ошибка 403 при редактировании
**Решение:**
- Это нормально, если пытаетесь редактировать не свое сообщение
- Редактировать можно только свои сообщения

### Проблема: Ошибка "Unknown column 'is_edited'"
**Решение:**
1. Миграция БД не применена
2. Выполните SQL команды из `001_add_message_edit_delete.sql`
3. Проверьте что таблица обновилась (`DESC chat_messages`)

### Проблема: Сообщение не удаляется
**Решение:**
1. Проверьте права доступа (только свои можно удалять для всех)
2. Посмотрите логи в консоли браузера (F12)
3. Посмотрите логи сервера Node.js
4. Проверьте что API endpoints добавлены в index.js

### Проблема: При обновлении страницы сообщение появляется обратно
**Решение:**
- Это нормально, сообщение было удалено только локально
- Нужно использовать "Удалить для всех" чтобы сообщение удалилось с сервера
