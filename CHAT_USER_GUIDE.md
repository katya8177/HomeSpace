# Как использовать редактирование и удаление сообщений

## Редактирование сообщения

1. **Наведите мышь** на свое сообщение в чате
2. **Нажмите на кнопку "⋮"** (три точки), которая появится справа
3. **Выберите "✎ Редактировать"**
4. **Внесите изменения** в появившемся диалоге
5. **Нажмите OK** для сохранения

После редактирования к сообщению будет добавлена отметка **(ред.)**

**Кто может редактировать:**

- ✅ Вы сами - свои сообщения
- ❌ Родители - не могут редактировать сообщения других

---

## Удаление сообщения

### Удалить только у себя

1. **Наведите мышь** на сообщение
2. **Нажмите на кнопку "⋮"** (три точки)
3. **Выберите "🗑 Удалить у себя"**
4. **Подтвердите** в появившемся диалоге

Сообщение исчезнет только из вашего чата, но останется у других членов семьи.

### Удалить для всех

1. **Наведите мышь** на сообщение
2. **Нажмите на кнопку "⋮"** (три точки)
3. **Выберите "🗑 Удалить для всех"**
4. **Подтвердите** в появившемся диалоге

Сообщение будет удалено для всех членов семьи.

**Кто может удалить для всех:**

- ✅ Автор сообщения
- ✅ Родители/Администраторы
- ❌ Остальные пользователи - только "Удалить у себя"

---

## Заметки

- Голосовые сообщения можно только удалять, но не редактировать
- При редактировании сохраняется оригинальный текст (в БД)
- Отметка "(ред.)" показывает, что сообщение было изменено
