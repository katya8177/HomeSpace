## 📋 Что было реализовано

### ✅ Редактирование сообщений

- Пользователь может редактировать **только свои** сообщения
- При редактировании к сообщению добавляется отметка **(ред.)**
- Сохраняется оригинальный текст в БД

### ✅ Удаление сообщений (две опции)

1. **Удалить у себя** - сообщение скрыто только для текущего пользователя
2. **Удалить для всех** - сообщение удалено из всей семьи

### ✅ Управление правами доступа

- **Обычный пользователь (user/child):** может редактировать/удалять только свои сообщения
- **Родитель/Администратор:** может удалять любые сообщения семьи, редактировать только свои

### ✅ Интерфейс

- Кнопка меню "⋮" (три точки) при наведении на сообщение
- Всплывающее меню с опциями (Редактировать, Удалить у себя, Удалить для всех)
- Интуитивное управление правами

## 🐛 Известные ограничения

- Редактирование голосовых сообщений не поддерживается (только удаление)
- История редактирования хранит только последнюю версию
- Нельзя восстановить удаленные сообщения без администратора БД
