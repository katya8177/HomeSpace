## Шаг 5: Тестирование функционала

### Редактирование сообщения
- [ ] Кнопка "⋮" появляется при наведении на свое сообщение
- [ ] Меню содержит опцию "✎ Редактировать"
- [ ] Можно отредактировать текст сообщения
- [ ] К сообщению добавляется отметка "(ред.)"
- [ ] Отметка видна всем пользователям
- [ ] Нельзя редактировать чужие сообщения (кнопка не появляется)

### Удаление для себя
- [ ] Кнопка "⋮" появляется на всех сообщениях
- [ ] Меню содержит опцию "🗑 Удалить у себя"
- [ ] Сообщение исчезает в моем чате
- [ ] Сообщение остается видно в чатах других пользователей
- [ ] При перезагрузке страницы сообщение появляется обратно (так как удалено только локально)

### Удаление для всех (обычный пользователь)
- [ ] На своем сообщении кнопка "🗑 Удалить для всех" активна (белая)
- [ ] На чужом сообщении кнопка "🗑 Удалить для всех" неактивна (серая)
- [ ] При нажатии на активную кнопку появляется подтверждение
- [ ] После подтверждения сообщение исчезает у всех
- [ ] При перезагрузке страницы сообщение не появляется обратно

### Удаление для всех (родитель/админ)
- [ ] На любом сообщении кнопка "🗑 Удалить для всех" активна (белая)
- [ ] Родитель/админ может удалить любое сообщение в семье
- [ ] После удаления сообщение исчезает для всех

### Проверка прав доступа
- [ ] Пользователь A не может редактировать сообщение пользователя B
- [ ] Пользователь A может удалить только свое сообщение (для всех)
- [ ] Родитель может удалить любое сообщение
- [ ] Ребенок может удалить только свои сообщения (для всех)

2. **Проверить логи сервера Node.js**
