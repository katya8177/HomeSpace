# Готовые корпуса комнат (Rooms)

Эта папка содержит готовые корпуса комнат для редактора мебели.

## Структура файлов

Каждая комната должна иметь подпапку с изображением в формате PNG. Для корпусов комнат используется только один фиксированный ракурс:

```
roomKitchen/
  └── roomKitchen_NE.png    # Фиксированная ориентация (северо-восток)
```

## Доступные комнаты

- `roomKitchen` - Кухонный гарнитур
- `roomBathroom` - Ванная комната
- `roomBedroom` - Спальня
- `roomLivingroom` - Гостиная
- `roomOffice` - Кабинет
- `roomKids` - Детская

## Добавление новых изображений

1. Создайте подпапку с названием комнаты (например, `roomKitchen`)
2. Добавьте PNG файл с суффиксом `_NE.png`:
   - `_NE.png` - Северо-восток (единственный ракурс для корпусов комнат)
   - `_SW.png` - Юго-запад (опционально)

- Комнаты отображаются в категории "🏠 Комнаты" в редакторе
- Размеры комнат указаны в конфигурации (src/config/furniture.js)
